School project
January 2017 - March 2017
Allows the user to vote on the topic chosen by the site administrator. Project made to pass IT in high school.
Own CMS
This website has its own CMS based on PHP and MySQL database prepared especially for it. The administrator can get to it via login page.
Many options
The administrator is not restricted by the rigid amount of possible answers thanks to the relational structure of the database. They can edit an existing survey and even change its parameters such as visibility for the ordinary user.
Voting without loging
The user can vote without logging in. Some data are stored in the database to prevent multiple voting. A tiny script blocks the ability to vote via proxy server.
Safe login page
It's a simple login form, but protected from possible SQL Injection attack.
