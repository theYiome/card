Projekt na własny użytek
maj 2017 - czerwiec 2017
Potrzeba wyróżnienia się na rynku pracy z pośród niezliczonej ilości początkujących programistów zmusiła mnie do stworzenia tego projektu.
Material Design
Stworzenia szaty graficznaj strony zainspirowane było designem zaproponowanym przez Google.
Responsive Web Design
Wygląd strona dostosowuje się do rozdzielczości okna przeglądarki aby zapewnić użytkownikowi należytą wygodę na dowolnym urządzeniu. Jest to możliwe dzięki zastowsowaniu Media Queries oraz jednostek stanowiących procent szerokości lub wysokości okna przeglądarki.
jQuery
Ta część strony została pobrana z serwera asynchronicznie aby zapewnić najlepszą wydajność. Wysuwany panel oraz formularz kontaktowy wysłający wiadomość asynchronicznie egzystuje dzięki pomocy wspaniałego frameworka do JavaScript jakim jest jQuery.
Icon Font
To nie są obrazki lecz czcionka. W tym projekcie znaleść można Font Awesome.
